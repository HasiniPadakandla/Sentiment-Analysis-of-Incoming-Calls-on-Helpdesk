Sentiment Analysis of Incoming Calls on Helpdesk focuses on analysing the sentiment of incoming calls on the helpdesk raised by the customer. It helps in increasing the customer satisfaction.

Features of the project :
1. Data collection - Efficiently gathers and preprocesses data from incoming helpdesk calls to ensure high-quality input for analysis.
2. Automated Speech Recognition - Converts spoken language from calls into text automatically, enabling further analysis.
3. Sentiment Analysis - uses VADER ALGORITHM
4. Visualization Dashboard - Display the results for sentiment analysis
